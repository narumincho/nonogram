# nonogram

https://nonogram.narumincho.com/

お絵かきロジック(イラストロジック, ノノグラム)を解いてくれるアプリです

![image](image/image.png)

- 途中まで解いたところから, 解かせることができます
- 埋められる数が多い順に埋めていく

のが特徴です

不成立な問題など, たまに解けない問題もあります.

成立する問題なのに解けない場合など問題点があれば issues で報告してもらえると助かります
